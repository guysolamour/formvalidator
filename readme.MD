Form Validator

Usage:
1- Required
add data-req='required' to the field

2- Email
add data-type='email' to the field

3-Phone number
add data-type='number' to the field

3- Minimum
add data-min='6'

4- Maximum
add data-max='15'

5- Password
add data-same='#password' to the verification field
the value must be the id of the first password field
